# 파이썬 디스코드 봇 "뽀삐"
### 목적 : 디스코드 봇 여러 기능 모음 (파일별 분류)
### 파일 분류
- main.py : 실행
- help.py : 도움말, 시간
- game_ ... .py : 게임 관련 기능
- leagueoflegend.py : 롤 API
- song.py : 노래 (pytube, etc)
- system.py : bot, token, api_key 등
### 주의사항
 본 코드는 모두 참고용입니다.   
 system.py의 bot, token, api_key는 None값 처리되어있습니다. (사용 시, 수정 필요)
